1.Введение

При изучении любого языка программирования, программисты традиционно пишут программу, которая выводит сообщение: "Hello, World!". Мы также придерживаемся этой традиции. Чтобы вывести что-то на экран в языке Python, используется функция print(): print("Hello, World!")
# Hello, World!

Наша программа успешно вывела сообщение: "Hello, World!". Для удобства, в комментарии к коду указано то, что вывела программа.

Для вывода на экран текста (строк) можно использовать двойные или одинарные кавычки ("Hello", 'Hello').
В Python это никак не повляет на результат.

Функция print() позволяет выводить различные данные на экран, например: текст, числа, значения переменных, о которых чуть более углубленно мы поговорим в следующем уроке.

О комментариях

Почти во всех языках прогроммирования есть возможность оставлять комментарии в коде. Python не был обделён.

Комментарий никак не используется интерпритатором, он предназначен для людей. Его используют, чтобы оставить какие-либо пояснения о исправлении кода, о его дополнении и т.д. Например: # Удалить строку ниже
print(10)

Комментарии в Python начинаются со знака #, их можно поместить в любом месте программы.
# Выводит "Привет"
print("Привет")
print(100) # Выводит 100

Функция input()

В процессе разработки какой-то программы, может возникнуть потребность получить данные от пользователя. Это весьма просто реализовать с помощью функции input(): name = input("Как тебя зовут? ")
# Выводит "Как тебя зовут? "
# После можно ввести какой-то текст, например введем Паша
 print("Привет, ", name)
# Выводит "Привет, Паша"

Описание

На этом курсе вы изучите необходимые для экзаменов основы языка Python.
Вы познакомитесь с синтаксисом Python, условиями, циклами, функциями, типами данных и некоторыми библиотеками, которые вам пригодятся при подготовке к ЕГЭ. После прохождения этого курса, у вас будет достачно знаний, чтобы начать готовиться к экзаменам.

В этом курсе помимо теории есть и практика в виде тестов и задач. Тесты помогут закрепить несложную информацию, а задачи помогут лучше погрузиться в язык.

Курс направлен на учеников 8-11 классов, которые хотят использовать Python в решении задач на Государственной Итоговой Аттестации.

Почему Python

  • Python - современный язык, очень прост в понимании, весьма краток.
  • Его функционал легко расширяется с помощью библиотек и модулей.
  • Многие считают Python лучшим языком для новичков.

Формат обучения

  1. Теория в виде статьи на определенную тему.
  2. Различные тесты.
  3. Практика в конце каждой темы в виде задач.
